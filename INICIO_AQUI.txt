## 🚀 PASOS PARA EMPEZAR

### PASO 1: Instalar Dependencias (Una sola vez)

**Opción A: Script automático (Windows)**
```bash
setup.bat
# Selecciona opción 1: Instalar dependencias
```

**Opción B: Script Python (Multiplataforma)**
```bash
python instalar.py
# Selecciona la opción que desees (1 para CRÍTICOS, 6 para TODO)
```

**Opción C: Manual (si las anteriores no funcionan)**
```bash
# Instalar librerías matemáticas
python -m pip install scipy numpy pandas pulp python-dotenv

# Instalar LangChain
python -m pip install langchain langchain-community langchain-groq

# Instalar ChromaDB y embeddings
python -m pip install chromadb sentence-transformers langchain-chroma

# Instalar PDF
python -m pip install pymupdf pypdf

# Instalar UI
python -m pip install streamlit langgraph
```

### PASO 2: Verificar Sistema

```bash
python verificador.py --diagnose
```

Deberías ver ✅ en:
- PDF (Libro Hillier)
- ChromaDB
- API Groq  
- Herramientas Matemáticas

**Si algo falla:**
```bash
# Regenerar ChromaDB
python verificador.py --regenerate-db
```

### PASO 3: Ejecutar la Aplicación

**Opción A: Interfaz Web (RECOMENDADA)**
```bash
streamlit run app_ui.py
```
Se abrirá en `http://localhost:8501`

**Opción B: Script de Prueba**
```bash
python prueba.py
```

## 🔧 TROUBLESHOOTING

### ❌ "ModuleNotFoundError: No module named '...'"
```bash
# Instalar todos los paquetes
python instalar.py
# Selecciona opción 6 (Instalar TODO)
```

### ❌ "GROQ_API_KEY no configurada"
1. Verifica que existe el archivo `.env`
2. Contiene: `GROQ_API_KEY=tu_api_key_de_groq`
3. Reinicia la aplicación

### ❌ "ChromaDB no disponible"
```bash
python verificador.py --regenerate-db
```

### ❌ "Streamlit no inicia"
```bash
# Limpiar caché
streamlit cache clear

# Ejecutar con debug
streamlit run app_ui.py --logger.level=debug
```

## 🎓 EJEMPLO COMPLETO DE USO

### Escenario: Problema de Producción

**Problema:**
```
Una empresa fabrica mesas (M1) y sillas (M2).
- Ganancias: M1=$80, M2=$60 por unidad
- Madera disponible: 300 unidades
- Horas trabajo: 240 horas
- Requerimientos:
  * M1: 6 madera, 8 horas
  * M2: 8 madera, 6 horas
¿Plan óptimo?
```

**Lo que hace el sistema:**
1. ✅ Clasifica como "Programación Lineal Continua"
2. ✅ Busca teoría del Método Simplex
3. ✅ Formula el problema matemáticamente
4. ✅ Usa SciPy linprog para resolver
5. ✅ Retorna: "Producir X mesas y Y sillas para ganancia máxima de $Z"

**Resultado esperado:**
```
✅ ÓPTIMO ENCONTRADO
Solución: {'x1': 30, 'x2': 12.5}
Valor Objetivo: 3150.00
```
